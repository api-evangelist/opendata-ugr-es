# Bienvenido a OpenData UGR (opendata-ugr-es)

Bienvenido a OpenData UGR is a open data portal open-data portal for Spain running **CKAN**, exposing a standardized catalog API over ~65 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/opendata-ugr-es/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=opendata-ugr-es-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.5

## API
- **Bienvenido a OpenData UGR CKAN Action API** — base `https://opendata.ugr.es/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://opendata.ugr.es)
- Live endpoint verified: `https://opendata.ugr.es/api/3/action/status_show` · ~65 datasets
